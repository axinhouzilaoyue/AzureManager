              请先点击“检查账户”。通过检查后才允许保存。
            

          

          

            不知道这些值从哪里获取？展开查看 Azure CLI / Portal 指引
            

              `subscriptionId` 和 `tenantId` 属于你自己的 Azure 订阅信息，不会也不应该从 GitHub 仓库里推断出来。
            

            
# 1) 登录 Azure
az login

# 2) 查看当前订阅和租户
az account show --query "{subscriptionId:id, tenantId:tenantId, subscriptionName:name}" --output json

# 3) 为当前订阅创建本项目使用的服务主体
SUBSCRIPTION_ID="$(az account show --query id --output tsv)"
az ad sp create-for-rbac \
  --name "azure-panel-cf" \
  --role "Contributor" \
  --scopes "/subscriptions/$SUBSCRIPTION_ID"

            

              命令返回的 `appId` 填到“客户端 ID”，`password` 填到“客户端密码”，`tenant` 填到“租户 ID”；`subscriptionId` 则来自 `az account show` 或 Azure Portal 的 Subscriptions 页面。
