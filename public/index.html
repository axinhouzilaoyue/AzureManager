使用说明
快速上手指南

🚀 快速开始

  • 点击左侧「添加账户」，填入 Azure Service Principal 的 client_id、client_secret、tenant_id、subscription_id
  • 点击「检查凭据」验证账户有效性，通过后保存
  • 进入「账户管理」，点击账户卡片查看该订阅下的虚拟机

☁️ 创建虚拟机

  • 进入账户 → 点击「创建虚拟机」
  • 选择区域、实例规格、操作系统、磁盘大小
  • 可选填开机脚本，留空则使用「全局设置」中的默认脚本
  • 任务提交后可在「任务」标签页查看进度

🔧 VM 操作

在虚拟机列表中，每台 VM 支持以下操作：

  • 启动 / 停止 / 重启：控制 VM 运行状态
  • 换 IP：释放旧公网 IP 并分配新静态 IP
  • 删除：删除整个资源组（含 VM 及所有关联资源）

🔑 Azure 凭据准备

  • 在 Azure Active Directory 创建 App Registration
  • 生成 Client Secret
  • 在目标订阅的 IAM 中为该 App 赋予 Contributor 角色
  • 记录 Application (client) ID、Directory (tenant) ID、Subscription ID

💾 数据迁移

所有数据（包括账户凭据和加密密钥）存储在 Docker 挂载目录的 data/ 文件夹中。迁移只需将整个 data/ 目录复制到新机器即可。
